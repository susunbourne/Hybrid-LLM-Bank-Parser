# What's this?

An LLM-based personal finance app that extracts transactions from bank statements(usually PDF format), classifies what you put your money on, and forecast future spending with time-series modeling.

## Features

- Whole ETL pipeline, from messy real world raw data to tidy relational dataset.
- Transaction classification with LLM method & fallback control
- Monthly expense forecasting with ARIMA
- Interactive Streamlit dashboard
- Docker support for reproducible deployment

## Quick Start

### Using main.py

Just type in "python main.py"

### Using streamlit

Just typr in "streamlit run app.py"
